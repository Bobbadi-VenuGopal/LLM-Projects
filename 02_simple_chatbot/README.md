# Project 2: Simple Chatbot

## Overview
Build a conversational AI that maintains context across multiple turns of conversation.

## What You'll Learn
- Conversation history management
- Context window handling
- Chat formatting and prompts
- Building interactive applications

## Model Used
- **TinyLlama-1.1B-Chat** - Optimized for conversations

## How to Run

```bash
cd 02_simple_chatbot
python chatbot.py
```

## Features
- Maintains conversation history
- Handles context window limits
- Chat-style formatting
- System prompts for personality

## Example Conversation
```
User: Hello! What's your name?
Bot: Hello! I'm an AI assistant. How can I help you today?

User: Can you tell me a joke?
Bot: Why don't scientists trust atoms? Because they make up everything!
```
