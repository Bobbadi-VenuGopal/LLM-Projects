### Week 3: Advanced

## Progression Tips

1. **Complete projects in order** - Each builds on previous concepts
2. **Experiment** - Modify code to see what happens
3. **Read the code** - Understand how things work, don't just run it
4. **Combine techniques** - Try using techniques from multiple projects
5. **Build your own projects** - Apply what you've learned
