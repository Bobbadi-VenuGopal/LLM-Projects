# Project 7: Text Summarization

## Overview
Create a system that can summarize long documents into shorter, concise versions.

## What You'll Learn
- Text summarization techniques
- Extractive vs Abstractive summarization
- Length control
- Multi-document summarization

## Model Used
- **TinyLlama-1.1B-Chat** - For summarization tasks

## How to Run

```bash
cd 07_text_summarization
python summarization.py
```

## Features
- Short summaries (1-2 sentences)
- Medium summaries (paragraph)
- Long summaries (multi-paragraph)
- Extractive summarization (key sentences)
- Abstractive summarization (new text)

## Example
```
Original: [Long article about AI...]
Summary: "AI is transforming technology through machine learning and neural networks..."
```
