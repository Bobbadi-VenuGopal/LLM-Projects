# Project 11: Multi-Agent System

## Overview
Build a system where multiple AI agents work together to solve complex tasks.

## What You'll Learn
- Agent architecture
- Inter-agent communication
- Task delegation
- Agent specialization
- Coordination strategies

## Model Used
- **TinyLlama-1.1B-Chat** - For each agent

## How to Run

```bash
cd 11_multi_agent_system
python multi_agent.py
```

## Features
- Specialized agents (Researcher, Writer, Analyzer)
- Agent communication
- Task coordination
- Parallel processing

## Example
```
Task: "Research and write about AI"
- Researcher Agent: Gathers information
- Writer Agent: Creates content
- Analyzer Agent: Reviews and improves
```
