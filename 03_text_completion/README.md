# Project 3: Text Completion / Autocomplete

## Overview
Create an autocomplete system that suggests the next words or completes sentences.

## What You'll Learn
- Text completion techniques
- Prefix handling
- Real-time suggestions
- Different completion strategies

## Model Used
- **GPT-2 Small** (~500MB) - Great for completion tasks

## How to Run

```bash
cd 03_text_completion
python completion.py
```

## Features
- Complete sentences from partial input
- Generate multiple completion options
- Adjustable completion length
- Different sampling strategies

## Example
```
Input: "The weather today is"
Completions:
1. "The weather today is beautiful and sunny..."
2. "The weather today is quite cold and rainy..."
3. "The weather today is perfect for a picnic..."
```
