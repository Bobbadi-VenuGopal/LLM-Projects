# Project 6: Sentiment Analysis

## Overview
Analyze the emotional tone and sentiment of text using LLMs.

## What You'll Learn
- Sentiment classification
- Emotion detection
- Prompt-based classification
- Batch processing

## Model Used
- **TinyLlama-1.1B-Chat** - For sentiment analysis

## How to Run

```bash
cd 06_sentiment_analysis
python sentiment_analysis.py
```

## Features
- Positive/Negative/Neutral classification
- Emotion detection (happy, sad, angry, etc.)
- Confidence scores
- Batch analysis

## Example
```
Text: "I love this product! It's amazing!"
Sentiment: Positive (Confidence: 95%)
Emotion: Happy
```
